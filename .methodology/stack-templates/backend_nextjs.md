# [PC] Contexto de Proyecto – Backend Node.js + NestJS

## 1. Objetivo del proyecto

Este repositorio implementa servicios backend usando **Node.js** y **NestJS**, siguiendo una metodología:
- Orientada a **TDD** (primero tests, luego código).
- Con **documentación generada en parte por IA**, basada en plantillas humanas.
- Con **migración gradual de código legacy** hacia una arquitectura más moderna.

La IA participa principalmente en:
- Generación y mantenimiento de tests.
- Generación de código a partir de tests.
- Generación de documentación técnica (Markdown/LaTeX) y diagramas PUML.

El humano:
- Define el contexto de negocio.
- Valida que el comportamiento del sistema sea el correcto.

---

## 2. Stack tecnológico

- **Lenguaje:** TypeScript
- **Runtime:** Node.js
- **Framework backend:** NestJS
- **Gestor de paquetes:** pnpm
- **Base de datos:** SQLServer
- **ORM / query builder:** Interno de NestJS
- **Framework de testing:** Jest
- **Linter / estilo:** ESLint + Prettier
