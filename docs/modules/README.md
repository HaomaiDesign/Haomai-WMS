# Módulos del Proyecto Haomai WMS

Este documento sirve como índice general de los módulos del sistema Haomai WMS. Cada módulo contiene su propia documentación detallada en su respectiva carpeta.

## Índice de Módulos

### [Pedidos y Movimientos](./pedidos-movimientos/overview.md)
**Descripción:** Gestión integral de ingresos y egresos de mercadería. Permite registrar entradas con escáner, lotes y vencimientos, así como salidas asociadas a pedidos o clientes.
**Roles:** Empleado, Administrador.

### [Logística](./logistica/overview.md)
**Descripción:** Gestión de pedidos de distribución/venta y entregas. Permite planificar rutas y hacer seguimiento de las entregas.
**Roles:** Empleado, Administrador.

### [Depósito](./deposito/overview.md)
**Descripción:** Administración centralizada del inventario físico. Incluye gestión de lista de productos, consulta de disponibilidad, historial de registros y gestión de ubicaciones.
**Roles:** Empleado, Administrador.

### [Clientes](./clientes/overview.md)
**Descripción:** Gestión y administración de la base de datos de clientes, incluyendo su historial de pedidos y entregas.
**Roles:** Empleado administrativo/ventas, Administrador.

### [Reportes](./reportes/overview.md)
**Descripción:** Módulo de inteligencia de negocios para visualizar métricas, alertas de stock, vencimientos y reportes por categoría para la toma de decisiones.
**Roles:** Gerente, Administrador.
