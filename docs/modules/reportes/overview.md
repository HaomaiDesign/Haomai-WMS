# Módulo: Reportes

## Descripción
Módulo orientado a la inteligencia de negocios y soporte a la toma de decisiones. Agrega la información transaccional del sistema para presentar métricas, indicadores de rendimiento (KPIs) y alertas tempranas sobre el estado del inventario.

## Funcionalidades Principales
- **Reporte por Categoría:** Visión de stock agrupado por familias de productos para análisis de rotación.
- **Vencimientos:**
  - **Próximos a vencer:** Listado preventivo para acciones comerciales (FEFO).
  - **Caducados:** Identificación de pérdida/merma por vencimiento.
- **Alerta de Inventario:** Notificaciones de productos que han alcanzado su punto de pedido (stock mínimo).
- **Reporte de Movimiento de Stock:** Historial detallado y filtrable de ingresos y egresos para auditorías y control.

## Roles de Usuario
- **Gerente:** Usuario principal, utiliza la información para comprar, vender o ajustar estrategias.
- **Administrador:** Monitorea la salud operativa del depósito.

## Relación con otros módulos
- **Depósito:** Lee el estado actual del stock y ubicaciones.
- **Pedidos y Movimientos:** Analiza el flujo histórico de entradas y salidas.
