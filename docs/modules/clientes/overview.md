# Módulo: Clientes

## Descripción
Gestiona la información de las entidades externas (clientes) que interactúan con el sistema, principalmente como receptores de mercadería. Funciona como un maestro de clientes para facilitar la operación de ventas y logística.

## Funcionalidades Principales
- **Listado de Clientes:**
  - Alta, Baja y Modificación (ABM) de clientes.
  - Información de contacto, dirección de entrega y datos fiscales.
- **Historial:** Consulta rápida de pedidos y entregas realizadas a cada cliente.

## Roles de Usuario
- **Ventas / Administrativo:** Da de alta clientes y actualiza sus datos.
- **Administrador:** Gestión general.

## Relación con otros módulos
- **Pedidos y Movimientos:** Los egresos se asocian a un cliente seleccionado de este módulo.
- **Logística:** Provee los datos de destino para las entregas.
