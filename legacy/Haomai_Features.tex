\section*{Haomai WMS}
Haomai WMS (Warehouse Management System) es la solución ideal para controlar el flujo de mercadería en depósitos, simplificando y optimizando cada operación.

\begin{itemize}[leftmargin=*, itemsep=0.7em]
  \item \textbf{Ingreso y Egreso Ágil de Mercadería:} Registre entradas y salidas de productos de forma rápida, usando escáner de código de barras y eligiendo el depósito y lote correspondiente.
  \item \textbf{Control de Lotes y Vencimientos:} Asigne fechas de vencimiento al ingresar productos y seleccione el lote exacto al egresar, asegurando trazabilidad y rotación eficiente.
  \item \textbf{Inventario en Tiempo Real:} Consulte la disponibilidad actualizada de cada producto y evite descuadres gracias a la actualización automática del stock.
  \item \textbf{Gestión Multi-Depósito:} Administre varios depósitos y ubicaciones desde un solo sistema, optimizando el almacenamiento y la preparación de pedidos.
  \item \textbf{Reportes, Relatorios y Alertas Inteligentes:} Tome decisiones rápidas y precisas gracias a un sistema de reportes (relatorios) y alertas adaptado a cada perfil:
    \begin{itemize}
      \item \textbf{Para Gerencia:} Reportes de stock por categoría y movimientos para análisis y planificación.
      \item \textbf{Para Empleados:} Reportes de disponibilidad y ubicación para agilizar el trabajo diario.
      \item \textbf{Vencimientos y Alertas:} Listados de productos próximos a vencer, caducados y alertas de stock bajo.
    \end{itemize}
  \item \textbf{Trazabilidad Total:} Historial completo de movimientos por producto, lote y ubicación.
\end{itemize}

\vspace{1em}
\textbf{Haomai WMS le permite reducir errores, ahorrar tiempo y tener el control total de su inventario.}

\section*{Escalabilidad y Customización}
Haomai WMS está diseñado con una arquitectura modular, lo que permite su expansión y adaptación a medida que crecen las necesidades del negocio. Algunas posibilidades incluyen:

\begin{itemize}[leftmargin=*, itemsep=0.7em]
  \item \textbf{Agregar nuevos módulos:} Es posible incorporar funcionalidades adicionales, como módulos de e-commerce, gestión de mantenimiento, control de calidad, integración con balanzas, gestión documental, trazabilidad avanzada, o cualquier otro proceso específico del negocio.
  \item \textbf{Integraciones externas:} El sistema puede conectarse con sistemas de proveedores, transportistas o plataformas de ventas de terceros para automatizar procesos.
  \item \textbf{Customización de pantallas y reportes:} Cada pantalla, flujo o reporte puede adaptarse a los procesos y requerimientos específicos de cada empresa.
  \item \textbf{Soporte multi-empresa y multi-idioma:} El sistema puede configurarse para operar con varias empresas y en diferentes idiomas.
  \item \textbf{Perfiles y roles personalizados:} Es posible crear diferentes perfiles de usuario según roles y accesos, asignando a cada uno funcionalidades y permisos específicos según las necesidades del negocio.
\end{itemize}

Haomai WMS es una plataforma flexible y preparada para evolucionar junto a su negocio, permitiendo personalizaciones y expansiones a medida que cambian los desafíos logísticos y comerciales.
